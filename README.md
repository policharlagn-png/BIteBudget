# 🌿 BiteBudget

> AI-powered meal planning and grocery savings for families on a budget.

BiteBudget generates personalized 7-day meal plans based on your family size, dietary needs, fitness goals, and weekly budget — then finds real grocery stores near you and compares prices so you always know where to shop.

---

## Features

- **AI Meal Planner** — 7-day family meal plans with breakfast, lunch, dinner, and snacks generated by Llama 3.1 via Groq
- **Food Map** — Finds real nearby grocery stores ranked by nutrition score, filtered to groceries only
- **Shopping List** — Autocomplete search across 500+ branded grocery items with live local price comparison
- **My Personal Goals** — BMR/TDEE calculator with macro targets, calorie goals, and diet preferences
- **Price Compare** — Side-by-side price breakdown across stores near your zip code
- **Family Support** — Per-member dietary restrictions and activity levels, all respected in the meal plan

## Getting Started

### Prerequisites

- Node.js v18 or higher
- A Google Maps API key (with Maps JS + Places enabled)
- A Groq API key (free at [console.groq.com](https://console.groq.com))
- A USDA FoodData API key (free at [fdc.nal.usda.gov](https://fdc.nal.usda.gov))

### Installation

1. **Clone the repository**
```bash
   git clone https://github.com/yourusername/bitebudget.git
   cd bitebudget
```

2. **Set up environment variables**
```bash
   cp backend/.env.example backend/.env
```
   Fill in your keys in `backend/.env`:
```env
   GOOGLE_MAPS_API_KEY=your_key_here
   GROQ_API_KEY=your_key_here
   USDA_API_KEY=your_key_here
   PORT=3001
   NODE_ENV=development
```

3. **Run the app**
```bash
   bash start.sh
```
   Then open [http://localhost:5173](http://localhost:5173) in your browser.

   On macOS you can also double-click `START.command`.
